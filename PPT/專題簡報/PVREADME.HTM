使用 PowerPoint Viewer

關於 PowerPoint Viewer

Microsoft Office PowerPoint Viewer 能讓您檢視在 Microsoft Office PowerPoint 2000、PowerPoint XP、PowerPoint 2003，和 PowerPoint 2007 中建立的簡報。PowerPoint Viewer 也支援開啟受密碼保護的 PowerPoint 簡報。

注意：若要檢視在 PowerPoint 2007 中建立的簡報，您需要安裝 Office 2007 Word、Excel 和 PowerPoint 檔案格式的 Microsoft Office 相容性套件。請參閱 http://office.microsoft.com/zh-tw。

系統需求

  • 電腦的處理器需為 200 MHz 以上
  • Microsoft Internet Explorer 5 或更新版本
  • 支援的作業系統：包含 Service Pack 4 的 Microsoft Windows 2000、包含 Service Pack 1 的 Windows XP、Windows Server 2003，或 Windows Vista。
  • 隨機存取記憶體 (RAM)：
    • 包含 Service Pack 4 的 Microsoft Windows 2000：64 MB RAM
    • 包含 Service Pack 1 的 Windows XP、Windows Server 2003：128 MB RAM
    • Windows Vista：512 MB RAM
  • 5 MB 的硬碟可用空間
  • Super VGA (800 x 600) 以上的螢幕解析度

多語言問題

PowerPoint Viewer 不支援 Microsoft Office 校訂工具。

PowerPoint Viewer 不支援在 PowerPoint 97 中建立的雙向語言簡報。

使用 PowerPoint Viewer

PowerPoint Viewer 無法正確顯示圖表動畫。將以完整圖表代替動畫的方式顯示。

PowerPoint Viewer 不支援資訊授權管理 (IRM) 簡報。

PowerPoint Viewer 不支援正在執行的巨集、程式，或開啟連結或內嵌的物件。

PowerPoint Viewer 不支援顯示簡報中的以下圖形格式連結：.eps、.pct、.emz、.cdr、.cgm、.wpg，和 .wmz。
